# 🍰 Sprinkle and Sweets Bakery

A simple **bakery-themed website** built with **HTML, CSS, and JavaScript**.  
This is a local version of the website, designed for showcasing bakery products and creating a delightful browsing experience.

- Fully responsive design: every single page adapts seamlessly to **desktop, tablet, and mobile devices**
- Bakery-themed design with sweets and dessert visuals
- Interactive elements using JavaScript
- Purely front-end (no backend or database)
